TYDES

Bill and Cash Flow Tracking With Plaid Connectivity

TYDES is an iOS application for connecting financial accounts, tracking liabilities and due dates, and planning near-term cash flow.

Privacy Policy Privacy Choices GitHub Repository

What The App Does

  • Creates Plaid Link tokens from a secure backend service for authorized financial connections.
  • Requires explicit app consent before Plaid Link access.
  • Supports Plaid OAuth and Universal Link return for supported institutions.
  • Shows Dashboard, Accounts, Schedule, and Cash Flow views for balances, due dates, statement status, and near-term planning.

Security and Data Handling

  • Network requests use HTTPS/TLS.
  • Authentication uses Apple Sign In and backend session controls.
  • Sensitive Plaid data stored by the backend is encrypted at rest.
  • The backend is scoped to liabilities and authorized checking/cash balance data for Cash Flow; general account and transaction-list endpoints remain disabled.
