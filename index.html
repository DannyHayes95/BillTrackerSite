Bill Tracker Hub

Liability Tracking With Plaid Connectivity

Bill Tracker Hub is an iOS application for connecting financial accounts and viewing liability balances, amounts due, and due-date status.

Privacy Policy Privacy Choices GitHub Repository

What The App Does

  • Creates Plaid Link tokens from a secure backend service.
  • Supports Plaid OAuth and Universal Link return for supported institutions.
  • Shows liabilities with due-date indicators in the iOS client.

Security and Data Handling

  • Network requests use HTTPS/TLS.
  • Authentication uses Apple Sign In and backend session controls.
  • Plaid access tokens are encrypted at rest.
  • The backend is liabilities-focused and does not expose account/transaction feeds.
